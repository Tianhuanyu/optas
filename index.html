OpTaS Documentation

OpTaS is an OPtimization-based TAsk Specification library for task and motion planning (TAMP), trajectory optimization, and model predictive control.

Requirements

  • Numpy
  • Scipy
  • matplotlib
  • CasADi
  • urdf-parser-py
  • osqp
  • cvxopt
  • xacro
  • PyBullet (to run examples)
